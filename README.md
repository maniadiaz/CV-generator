# CV Generator - Formato Harvard

Aplicacion para generar CVs profesionales con formato Harvard. Guarda los datos localmente y permite exportar a PDF.

## Caracteristicas

- Formato Harvard para CV profesional
- Guardado automatico en localStorage
- Multiples perfiles de usuario
- Exportar/Importar datos en JSON
- Generacion de PDF
- Secciones:
  - Datos personales y redes sociales
  - Portafolio personal
  - Experiencia laboral
  - Educacion
  - Habilidades
  - Idiomas
  - Certificaciones

## Instalacion

```bash
npm install
```

## Uso

```bash
npm run dev
```

Abre http://localhost:5173 en tu navegador.

## Build

```bash
npm run build
```

## Tecnologias

- React + Vite
- Tailwind CSS
- jsPDF (generacion de PDF)
- React Router
- Lucide React (iconos)
